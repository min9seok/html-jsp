[뷰포트(viewport)]
  1. 웹 페이지에서 사용자가 화면상에 볼 수 있는 영역
  2. 여러 디바이스(스마트폰, 테블릿, 컴퓨터 등등) 표시 영역 다르다.
  3. 여러 장치의 크기에 맞게 웹 페이지의 크기를 제어해서 화면에 보여 줄 필요가 있다.
  4. 뷰포트 설정

  width=device-width 너비는 장치의 맞게
  initial-scale=1.0 비율 1.0 조정 ( 축소, 확대) 

  [미디어쿼리(Media Queries)]
  - css3 추가 기술
  - 여러 장치 마다 서로 다른 스타일을 적용할 필요 O
  - 반응협 웹
  - @media 규칙
  - 1) 미디어 쿼리 형식
    @media 조건식([참]/거짓){
    	css코딩;
    }
    @media [only|not 미디어타입 and] (조건식 참/거짓){
    	css코딩;
    }
    미디어타입 : [all] screen, print 등등
  - 2) link 태그
    예) print.css 외부 스타일 시트 파일
