[ img ]
 - html > images 폴더 생성
 - 웹페이지에 이미지 추가( 웹 페이지의 디자인)
 - 웹 페이지에 이미지 삽입 X,
   단지 이미지가 웹페이지에 연결
 - 2가지 필수 속성 지정
   src - 이미지 경로
   alt - 이미지 대체할 문자열
 - width ( 너비 ) height ( 높이 ) 를 지정하지 않으면 원래 이미지의 너비 높이로 설정
 - 항상 이미지의 너비, 높이 속성 지정 
    너비 높이가 지정되지 않으면 이미지가 로딩되는 동안 웹 페이지가 깜박임 발생하기 때문에
    
 - 이미지는 인라인 모드 이다.
